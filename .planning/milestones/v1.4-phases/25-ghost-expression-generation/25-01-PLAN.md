**Builder functions (per D-01):**

1. `build-reference (name &optional qualifiers)` -- Constructs `@name` or `@name{key=val, key2=val2}`. `qualifiers` is an alist of `(key . value)` string pairs. Use `(format nil "@~a" name)` as base, append `{k=v, ...}` if qualifiers present.

2. `build-commission (agent-id action)` -- Constructs `(agent-id){action}`. Use `(format nil "(~a){~a}" agent-id action)`. Per D-11, agent-id uses DB id column (e.g., "nova", "kathryn").

3. `build-search (search-type &optional qualifiers)` -- Constructs `![type]` or `![type]{key=val}`. Use `(format nil "![~a]" search-type)` base with optional qualifier block.

4. `build-bundle (name)` -- Constructs `{name}`. Use `(format nil "{~a}" name)`.

**Validation function (per D-03):**

5. `validate-innate-expression (expr)` -- Parse-round-trip validation. Returns `T` if valid, `(values nil error-string)` if invalid. Implementation:
```lisp
(defun validate-innate-expression (expr)
  (handler-case
      (let ((ast (parse (tokenize expr))))
        (if ast t (values nil "Parser returned empty result")))
    (innate-parse-error (e)
      (values nil (format nil "Parse error: ~a" e)))
    (error (e)
      (values nil (format nil "Unexpected error: ~a" e)))))
```
Import `innate-parse-error` from `:innate.conditions`.

**Slug generation (per D-07):**

6. `name-to-slug (name)` -- Kebab-case conversion. Lowercase, replace spaces/underscores/dots with hyphens, strip other non-alphanumeric chars. Implementation:
```lisp
(defun name-to-slug (name)
  (string-downcase
   (with-output-to-string (s)
     (loop for ch across name
           do (cond
                ((alphanumericp ch) (write-char ch s))
                ((or (char= ch #\Space) (char= ch #\_) (char= ch #\.))
                 (write-char #\- s)))))))
```

**Template CRUD (per D-05, D-06):**

7. `db-insert-template (name body &key category description parameters)` -- INSERT INTO templates. Use `db-escape` for ALL string values. Derive slug from name via `name-to-slug`. Handle `NULL` for optional fields. Wrap in handler-case to catch unique slug violations. On slug collision, append "-2" suffix and retry once. Returns T on success, (values nil error-string) on failure.
```lisp
(defun db-insert-template (name body &key category description parameters)
  (let* ((slug (name-to-slug name))
         (sql (format nil
                "INSERT INTO templates (name, slug, category, description, body, parameters) VALUES (~a, ~a, ~a, ~a, ~a, ~a)"
                (db-escape name) (db-escape slug)
                (if category (db-escape category) "NULL")
                (if description (db-escape description) "NULL")
                (db-escape body)
                (if parameters (db-escape (encode-json parameters)) "'[]'::jsonb"))))
    (handler-case (progn (db-execute sql) t)
      (error (e)
        (if (search "duplicate key" (format nil "~a" e))
            (handler-case
                (let ((retry-sql (format nil
                                   "INSERT INTO templates (name, slug, category, description, body, parameters) VALUES (~a, ~a, ~a, ~a, ~a, ~a)"
                                   (db-escape name) (db-escape (concatenate 'string slug "-2"))
                                   (if category (db-escape category) "NULL")
                                   (if description (db-escape description) "NULL")
                                   (db-escape body)
                                   (if parameters (db-escape (encode-json parameters)) "'[]'::jsonb"))))
                  (db-execute retry-sql) t)
              (error (e2) (values nil (format nil "INSERT failed: ~a" e2))))
            (values nil (format nil "INSERT failed: ~a" e)))))))
```

8. `db-update-template-body (template-id new-body)` -- UPDATE templates SET body WHERE id. Use `db-escape` for new-body. The DB trigger `trg_template_version_history` fires automatically on UPDATE. Returns T on success, (values nil error-string) on failure.
```lisp
(defun db-update-template-body (template-id new-body)
  (handler-case
      (let ((sql (format nil "UPDATE templates SET body = ~a WHERE id = ~a"
                   (db-escape new-body) template-id)))
        (db-execute sql) t)
    (error (e)
      (values nil (format nil "UPDATE failed: ~a" e)))))
```

9. `db-find-template-by-name (name)` -- SELECT id, name, slug, body FROM templates WHERE LOWER(name) = LOWER(escaped). Returns hash-table or nil. Used by Plan 02 for template updates.
```lisp
(defun db-find-template-by-name (name)
  (handler-case
      (let* ((sql (format nil "SELECT id, name, slug, body FROM templates WHERE LOWER(name) = LOWER(~a) LIMIT 1" (db-escape name)))
             (results (db-query sql)))
        (when (and results (> (length results) 0))
          (aref results 0)))
    (error () nil)))
```
