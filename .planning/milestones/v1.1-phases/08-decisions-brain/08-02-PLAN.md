```lisp
(defun extract-decision-lines (content)
  "Extract all DECISION: prefixed lines from content.
   Returns list of (decision . rationale) pairs.
   Parses 'DECISION: X because Y' into separate decision and rationale."
  (let ((lines (uiop:split-string content :separator '(#\Newline)))
        (results nil))
    (dolist (line lines)
      (let* ((trimmed (string-trim '(#\Space #\Tab) line))
             (prefix "DECISION:")
             (pos (search prefix trimmed)))
        (when (and pos (= pos 0))
          (let* ((rest (string-trim '(#\Space) (subseq trimmed (length prefix))))
                 (because-pos (search " because " rest)))
            (if because-pos
                (push (cons (string-trim '(#\Space) (subseq rest 0 because-pos))
                            (string-trim '(#\Space) (subseq rest (+ because-pos 9))))
                      results)
                (push (cons rest nil) results))))))
    (nreverse results)))
```

This implements the discretion recommendation to parse "DECISION: X because Y" into decision + rationale fields.
