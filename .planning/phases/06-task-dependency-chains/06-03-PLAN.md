```lisp
(defun parse-create-task-lines (content)
  "Extract CREATE_TASK: <description> assignee=<agent_id> blocked_by=#id,#id lines.
   Returns list of (description assignee-or-nil blocked-by-ids-or-nil)."
  (let ((results '())
        (lines (uiop:split-string content :separator '(#\Newline))))
    (dolist (line lines)
      (let ((trimmed (string-trim '(#\Space #\Tab #\Return) line)))
        (when (and (> (length trimmed) 12) (search "CREATE_TASK:" trimmed))
          (let* ((after (string-trim '(#\Space #\Tab)
                         (subseq trimmed (+ (search "CREATE_TASK:" trimmed) 12))))
                 (assignee-pos (search "assignee=" after))
                 (blocked-by-pos (search "blocked_by=" after))
                 ;; Description ends at whichever key=value comes first
                 (desc-end (or (when (and assignee-pos blocked-by-pos)
                                 (min assignee-pos blocked-by-pos))
                               assignee-pos
                               blocked-by-pos
                               (length after)))
                 (description (string-trim '(#\Space #\Tab)
                               (subseq after 0 desc-end)))
                 (assignee (when assignee-pos
                             (let* ((start (+ assignee-pos 9))
                                    (end (or (when blocked-by-pos
                                               (and (> blocked-by-pos start) blocked-by-pos))
                                             (position #\Space after :start start)
                                             (length after))))
                               (string-trim '(#\Space #\Tab)
                                 (subseq after start end)))))
                 (blocked-by-str (when blocked-by-pos
                                   (let* ((start (+ blocked-by-pos 11))
                                          (end (or (when assignee-pos
                                                     (and (> assignee-pos start) assignee-pos))
                                                   (position #\Space after :start start)
                                                   (length after))))
                                     (subseq after start end))))
                 (blocked-by-ids (when blocked-by-str
                                   (remove nil
                                     (mapcar (lambda (s)
                                               (parse-integer (string-trim '(#\# #\Space) s)
                                                              :junk-allowed t))
                                             (uiop:split-string blocked-by-str
                                                                :separator '(#\,)))))))
            (when (and description (> (length description) 0))
              (push (list description assignee blocked-by-ids) results))))))
    (nreverse results)))
```

Key points:
- `blocked_by=` uses same `search` pattern as `assignee=` (per D-07)
- Description ends at whichever key=value param comes first
- Each key=value param's end is bounded by the other param or a space
- `#123,#456` is split by comma, each element has `#` stripped, parsed to integer
- `(third result)` is a list of integers or NIL
