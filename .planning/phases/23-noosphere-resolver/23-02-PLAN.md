**1. Agent ID resolution helper (for deliver-commission, per D-12):**
```common-lisp
(defun resolve-agent-id (name)
  "Resolve agent name to agent ID string. Case-insensitive match on id or full_name.
Returns the agent id string or nil."
  (handler-case
      (let* ((esc (db-escape (string-downcase name)))
             (sql (format nil
                    "SELECT id FROM agents WHERE LOWER(id) = LOWER(~a) ~
                     OR LOWER(full_name) = LOWER(~a) LIMIT 1"
                    esc esc))
             (results (db-query sql)))
        (when (> (length results) 0)
          (gethash :id (aref results 0))))
    (error (e)
      (format t "  [resolve-agent-id] error: ~a~%" e)
      nil)))
```

```common-lisp
(defmethod resolve-context ((r noosphere-resolver) context verb args)
  ;; Phase 23: return raw structure. Phase 24 will add template evaluation.
  (make-innate-result
   :value (list :context (if (stringp context) context (format nil "~a" context))

   :context :query))
```

Note: `tokenize` and `parse` are imported from `innate.parser.tokenizer` and `innate.parser` respectively via the package definition.
