```lisp
(defun format-innate-value (val)
  "Convert an Innate evaluation result to LLM-friendly string."
  (cond
    ((null val) "")
    ((stringp val) val)
    ((numberp val) (princ-to-string val))
    ;; Plist from resolver (e.g., (:ID 5 :NAME "Alpha" :STATUS "blocked"))
    ((and (listp val) (keywordp (first val)))
     (with-output-to-string (s)
       (loop for (k v) on val by #'cddr
             do (format s "~a: ~a  " (string-downcase (symbol-name k)) v))))
    ;; List of items (search results)
    ((listp val)
     (format nil "~{~a~^~%~}" (mapcar #'format-innate-value val)))
    (t (princ-to-string val))))
```

**2. evaluate-template-for-project** — loads and evaluates a template by project name (per D-01, D-02, D-04, D-06, D-10, D-11):
```lisp
(defun evaluate-template-for-project (project-name)
  "Load and evaluate a template associated with PROJECT-NAME.
   Uses load-bundle from noosphere-resolver to fetch AST nodes,
   wraps in :program node, evaluates with *noosphere-resolver* as resolver.
   Returns evaluated string or nil if no template / error.
   Per D-03: returns nil if no template (additive, not required).
   Per D-06: uses :scope :query (read-only references, commissions still fire per D-07).
   Per D-10: handler-case catches errors, returns fallback string.
   Per D-11: innate-resistance produces inline error marker."
  (handler-case
      (when *noosphere-resolver*
        (let ((children (load-bundle *noosphere-resolver* project-name)))
          (when children
            (let* ((program (make-node :kind :program :children children))
                   (env (make-eval-env :resolver *noosphere-resolver*
                                       :scope :query))
                   (results (evaluate program env)))
              ;; Flatten results to human-readable string (Pitfall 5: evaluate returns a list)
              (format nil "~{~a~^~%~}"
                      (mapcar #'format-innate-value (remove nil results)))))))
    (innate-resistance (r)
      (format nil "[unresolved: ~a]" (resistance-condition-message r)))
    (error (e)
      (format nil "[Template evaluation error: ~a]" e))))
```
