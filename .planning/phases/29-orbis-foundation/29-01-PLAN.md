Extend parse-simple-yaml in yaml.lisp to support 2-level nested key-value sections (per D-01, D-07, D-09). The parser must handle three cases:

1. **Top-level scalar:** `key: value` (existing, no change)
2. **Top-level list:** `key:` followed by indented `- "items"` (existing, no change)
3. **Nested section (NEW):** `key:` (no inline value) followed by indented `subkey: value` pairs, where sub-values can be scalars OR lists (`subkey:` followed by deeper-indented `- "items"`)

Implementation approach:
- Track indentation level for each line (count leading spaces)
- When a key has no inline value AND the next non-empty line is indented with a colon (not a dash), create a nested hash-table
- Inside nested sections, handle both `subkey: value` (scalar) and `subkey:` + `- items` (list)
- Return to parent level when indentation decreases back to 0

Also extend serialize-simple-yaml to emit nested sections:
- When a value is a hash-table (not string or list), emit `key:\n` then for each sub-entry:
  - If sub-value is a list: emit `  subkey:\n` then `    - "item"\n` for each
  - If sub-value is a string: emit `  subkey: value\n`
- Maintain emission order: scalars first, then nested sections, then top-level lists (responsibilities last)

Implementation:
```lisp
(defun write-ghost-yaml (agent-id responsibilities)
  (handler-case
      (let* ((yaml-path (ghost-yaml-path agent-id))
             (temp-path (format nil "~a.tmp" (namestring yaml-path)))
             ;; Load existing or create new
             (ht (if (probe-file yaml-path)
                     (af64.utils.yaml:parse-simple-yaml
                      (uiop:read-file-string yaml-path))
                     (make-hash-table :test #'equal))))
        ;; Ensure id is set
        (setf (gethash "id" ht) agent-id)
        ;; Update responsibilities
        (setf (gethash "responsibilities" ht) responsibilities)
        ;; Write full YAML via serializer
        (with-open-file (s temp-path :direction :output :if-exists :supersede
                                     :if-does-not-exist :create)
          (write-string (af64.utils.yaml:serialize-simple-yaml ht) s))
        (rename-file temp-path yaml-path)
        t)
    (error (e)
      (values nil (format nil "YAML write failed for ~a: ~a" agent-id e)))))
```
