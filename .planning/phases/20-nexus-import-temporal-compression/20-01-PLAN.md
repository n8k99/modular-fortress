**`dedup.py`** (per D-01, D-02, D-03):
- Function `get_canonical_set(cursor) -> list[dict]` that runs this SQL to get the canonical conversation set:
  ```sql
  WITH backup AS (
    SELECT id, path, title, content,
      regexp_replace(path, '^.*/(\d{4}/\d{2}/)', '\1') AS rel_path
    FROM documents
    WHERE path LIKE 'Archive/backup-Nebulab/Eckenrode Muziekopname/Engineering/Nexus AI Chat Imports/%'
      AND path ~ '\d{4}-\d{2}-\d{2}'
  ),
  retired AS (
    SELECT id, path, title, content,
      regexp_replace(path, '^.*/(\d{4}/\d{2}/)', '\1') AS rel_path
    FROM documents
    WHERE path LIKE 'Archive/Retired Nebulab/04 Archives/01 Nexus AI Chat Imports/%'
      AND path ~ '\d{4}-\d{2}-\d{2}'
  ),
  canonical AS (
    SELECT id, path, title, 'backup' as source FROM backup
    UNION ALL
    SELECT r.id, r.path, r.title, 'retired' as source
    FROM retired r
    LEFT JOIN backup b ON r.rel_path = b.rel_path
    WHERE b.id IS NULL
  )
  SELECT * FROM canonical ORDER BY path
  ```
  Returns list of dicts with keys: id, path, title, source.

- Function `generate_audit_report(cursor) -> str` that produces a markdown report:
  - Total backup-Nebulab count, total Retired-Nebulab count
  - Overlap count (by relative path)
  - Retired-only entries (list each path)
  - Backup-only entries (list each path)
  - Final canonical count
  Save report to archives table as a single record: `source_type='dedup_audit'`, `title='Nexus Chat AI Deduplication Audit'`, content=the markdown report, `period_start='2023-12-21'`, `period_end='2025-06-10'`.

- Function `run_dedup(dry_run=False)`: Orchestrates get_canonical_set + generate_audit_report. If not dry_run, inserts the audit report. Prints summary stats to stdout.

- CLI: `if __name__ == '__main__':` with argparse, `--dry-run` flag.

**`import_archives.py`** (per D-04, D-05, D-06):
- Function `run_import(dry_run=False, batch_size=100)`:
  1. Call `get_canonical_set(cursor)` from dedup module
  2. Check already-imported: `SELECT source_id FROM archives WHERE source_type='chatgpt_import'`
  3. For each doc not yet imported:
     - Extract date via `re.search(r'(\d{4}-\d{2}-\d{2})', path)` -> period_start = period_end = that date
     - Extract title via `re.search(r'\d{4}-\d{2}-\d{2} - (.+)\.md$', path)` -> if no match, use basename
     - Fetch full content from documents table: `SELECT content FROM documents WHERE id = %s`
     - Determine trivial flag: `len(content or '') < 2000` (per D-08 discretion, 2000 char threshold)
     - INSERT into archives: `INSERT INTO archives (title, content, source_type, source_id, original_path, period_start, period_end, topic, tags, metadata) VALUES (%s, %s, 'chatgpt_import', %s, %s, %s, %s, %s, %s::jsonb, %s::jsonb) RETURNING id`
       - topic = extracted title (to be refined by LLM later; only topic/tags/metadata can be UPDATEd after INSERT due to immutability trigger)
       - tags = `'[]'`
       - metadata = `json.dumps({"source": "nexus_import", "trivial": trivial_flag, "dedup_source": source_label})`
     - Commit in batches of `batch_size`
  4. Log progress every 50 records: `"Imported {n}/{total} conversations"`

- CLI: `if __name__ == '__main__':` with argparse, `--dry-run` and `--batch-size` flags.

**`verify.py`**: Verification script with these SQL checks:
1. IMPORT-01: `SELECT count(*) FROM archives WHERE source_type='dedup_audit'` -> must be >= 1
2. IMPORT-02: `SELECT count(*) FROM archives WHERE source_type='chatgpt_import' AND period_start IS NOT NULL` -> must be >= 990

4. IMPORT-04: `SELECT count(*) FROM memories WHERE path LIKE 'Nexus/%' AND nova_memories IS NOT NULL` -> must be > 0
5. IMPORT-05: `SELECT count(*) FROM memories WHERE content LIKE '%## Nexus Imports%'` -> must be > 0

Function `verify_all() -> dict` returns pass/fail per requirement. CLI main prints results.

All scripts must use `sys.path.insert(0, str(Path(__file__).resolve().parent.parent))` at top to enable `from _db import get_cursor` and `from _config import PATHS` imports when run from the nexus-import directory.
