Create `gotcha-workspace/tools/nexus-import/link_notes.py` (per D-15, D-16, D-17):

- Function `get_daily_note_template(cursor) -> str`:
  - `SELECT content FROM documents WHERE path = 'Templates/Daily Note.md'`
  - Return the template content string. If not found, return a minimal default:
    ```
    # {date_formatted}\n\n---\n\n## Notes\n\n
    ```

- Function `substitute_template(template: str, date: datetime.date) -> str`:
  - Replace `{{date:YYYY-MM-DD}}` with `date.strftime('%Y-%m-%d')`
  - Replace `{{date:YYYY}}` with `date.strftime('%Y')`
  - Replace `{{date:MM}}` with `date.strftime('%m')`
  - Replace `{{date:DD}}` with `date.strftime('%d')`
  - Replace `{{date:dddd}}` with `date.strftime('%A')`
  - Replace `{{date:ddd}}` with `date.strftime('%a')`
  - Replace `{{date:MMMM}}` with `date.strftime('%B')`
  - Replace `{{time:HH:mm}}` with `00:00`
  - Use `re.sub(r'\{\{[^}]+\}\}', '', result)` to clear any remaining unrecognized template vars

- Function `build_nexus_section(conversations: list[dict]) -> str`:
  - Takes list of dicts with keys: archive_id, title, period_start
  - Builds markdown section:
    ```
    \n\n## Nexus Imports\n\n
    - [[{title}]] — ChatGPT conversation from {date}\n
    ```
  - One wikilink per conversation on that date

- Function `run_link_daily(dry_run=False)`:
  1. Get all imported archive conversations grouped by date:
     ```sql
     SELECT period_start, json_agg(json_build_object('id', id, 'title', title) ORDER BY title) as conversations
     FROM archives WHERE source_type = 'chatgpt_import'
     GROUP BY period_start ORDER BY period_start
     ```
  2. Get daily note template
  3. For each date:
     a. Check if daily note exists: `SELECT id, content FROM memories WHERE path = %s` with path = `Areas/N8K99Notes/Daily Notes/YYYY-MM-DD.md`
     b. If exists:
        - Check if content already contains `## Nexus Imports` — if so, skip (idempotent)
        - Otherwise, append nexus section: `UPDATE memories SET content = content || %s WHERE id = %s`
     c. If not exists (per D-16):
        - Generate from template: `substitute_template(template, date)`
        - Append nexus section to generated content
        - INSERT: `INSERT INTO memories (path, content, note_type, note_date, compression_tier) VALUES (%s, %s, 'daily', %s, 'daily')`
          - path = `Areas/N8K99Notes/Daily Notes/YYYY-MM-DD.md`
     d. Log: `"Linked {n} conversations to daily note {date}"`
  4. Commit in batches of 50

- Function `run_link_weekly(dry_run=False)`:
  1. Get all imported archive conversations grouped by ISO week:
     ```sql
     SELECT date_trunc('week', period_start) as week_start,
            to_char(period_start, 'IYYY') || '-W' || to_char(period_start, 'IW') as week_label,
            count(*) as conv_count,
            json_agg(json_build_object('id', id, 'title', title, 'date', period_start) ORDER BY period_start) as conversations
     FROM archives WHERE source_type = 'chatgpt_import'
     GROUP BY date_trunc('week', period_start), to_char(period_start, 'IYYY') || '-W' || to_char(period_start, 'IW')
     ORDER BY week_start
     ```
  2. For each week:
     a. Find weekly note: `SELECT id, content FROM memories WHERE path LIKE %s` with `Areas/N8K99Notes/Weekly Notes/{week_label}%`
     b. If exists and not already containing `## Nexus Imports`:
        - Build a summary section listing conversation count per day that week and selected titles
        - Append: `UPDATE memories SET content = content || %s WHERE id = %s`
     c. If not exists: skip (don't generate weekly notes from template — only daily notes get generated per D-16)
     d. Log: `"Linked {n} conversations to weekly note {week_label}"`

- Function `run_link_notes(dry_run=False)`:
  - `run_link_daily(dry_run)`
  - `run_link_weekly(dry_run)`

- CLI: argparse with `--dry-run`, `--daily-only`, `--weekly-only` flags.

Use `sys.path.insert(0, str(Path(__file__).resolve().parent.parent))` at top for imports.
