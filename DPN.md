# DPN.md — DragonPunk Noir Style Guide

> *"Where dragons cast shadows in neon light, and ancient magic runs through fiber optic veins."*

This is the canonical visual style guide for **DragonPunk Noir** — a fusion of film noir aesthetics, cyberpunk technology, and high fantasy elements, set in the world of Orbis (10,500 years of history, colony ships, uploaded minds, dragons as apex beings).

## 1. Core Visual Pillars

DragonPunk Noir rests on **five essential pillars**. Every image should embody at least three:

### 🌑 **Noir Foundation**
- **High contrast** — deep blacks, bright highlights, minimal midtones
- **Dramatic shadows** — figures partially obscured, mystery in darkness
- **Moral ambiguity** — visual tension, nothing purely bright or clean
- **Urban decay meets grandeur** — crumbling elegance, worn luxury
- **Rain and wet surfaces** — reflections everywhere, gleaming streets

### ⚡ **Cyberpunk Overlay**
- **Neon signage** — holographic, flickering, in-world languages
- **Augmented reality elements** — floating UI, data streams, HUD overlays
- **Dense urban verticality** — towers, catwalks, layered city depths
- **Tech embedded in everything** — cables, screens, implants
- **Corporate/guild presence** — logos, propaganda, surveillance

### 🐉 **Fantasy Integration**
- **Dragons as apex beings** — not pets, not monsters: ancient intelligences
- **Magic as technology** — runes that function like circuits, spell-code
- **Medieval architecture evolved** — gothic spires with antenna arrays, castle-factories
- **Mythic scale** — dragons, ancient structures, cosmic scope
- **10,500 years of layered history** — ruins beneath ruins beneath cities

### 🚀 **Orbis Worldbuilding**
- **Colony ship heritage** — generation ships, uploaded consciousnesses
- **Post-scarcity tensions** — abundance with artificial scarcity
- **Faction aesthetics** — each guild/corp has distinct visual language
- **Temporal depth** — technology from multiple eras coexisting
- **Mind/machine blur** — digital ghosts, backup minds, consciousness transfer

### 🎬 **Cinematic Quality**
- **Film grain optional** — adds texture, grounds the image
- **Composition rules** — rule of thirds, leading lines, framing
- **Depth of field** — selective focus, bokeh, layered planes
- **Color grading** — not raw, always processed with intent
- **Narrative implication** — every image tells a story mid-scene

## 2. Color Palette

### Primary Colors — The Noir Foundation
| Name | Hex | Usage |
|------|-----|-------|
| Void Black | `#0A0A0F` | Deep shadows, backgrounds, mystery |
| Smoke Grey | `#2D2D3A` | Midtone shadows, fog, stone |
| Bone White | `#E8E4DC` | Highlights, fog, dragon bone, eyes in darkness |

### Secondary Colors — The Cyberpunk Injection
| Name | Hex | Usage |
|------|-----|-------|
| Neon Cyan | `#00F5FF` | Tech glow, holograms, cold lighting |
| Magenta Pulse | `#FF2D95` | Neon signs, accent lighting, warning |
| Electric Purple | `#9D00FF` | Magic auras, high-tech elements |
| Toxic Green | `#39FF14` | Data streams, system interfaces, corruption |

### Accent Colors — The Fantasy Warmth
| Name | Hex | Usage |
|------|-----|-------|
| Dragon Gold | `#FFB800` | Flame, treasure, power, dragon eyes |
| Blood Crimson | `#8B0000` | Violence, passion, danger, old magic |
| Ember Orange | `#FF6B35` | Fire, warmth in cold scenes, hope |
| Ancient Bronze | `#CD7F32` | Artifacts, aged metal, history |

### Color Usage Rules
1. **70% Dark Tones** — Noir demands darkness as default
2. **20% Neon Accents** — Punctuate, don't saturate
3. **10% Warm Highlights** — Fantasy warmth breaks the cold
4. **Complementary tension** — Cyan/orange, magenta/green create visual friction
5. **Desaturate backgrounds** — Saturated elements pop against muted environments

## 3. Lighting Guide

Lighting is the soul of DragonPunk Noir. Master these techniques:

### 🌑 **Chiaroscuro** (Primary Technique)
The Renaissance technique perfected for noir — extreme contrast between light and dark.
```
Prompt fragment: "chiaroscuro lighting, single harsh light source, deep dramatic shadows, Renaissance-inspired contrast, face half-illuminated"
```
*Use for:* Character portraits, dramatic reveals, contemplative moments

### 💡 **Single Source / Hard Light**
One dominant light creates sculptural shadows with sharp edges.
```
Prompt fragment: "single source lighting, hard shadows, sharp shadow edges, dramatic noir lighting, spotlight effect"
```
*Use for:* Interrogation scenes, lone figures, power dynamics

### 🔵 **Neon Wash**
Colored light from signage bleeds across the scene, tinting everything.
```
Prompt fragment: "neon wash lighting, pink and cyan color spill, wet surfaces reflecting neon, cyberpunk glow, multiple colored light sources"
```
*Use for:* Street scenes, urban environments, establishing shots

### 🔥 **Practical Lighting + Magic**
Light sources visible in frame (lanterns, screens, dragon flame, spell effects).
```
Prompt fragment: "practical lighting, visible light sources, magical flame illumination, screen glow, mixed color temperature"
```
*Use for:* Interior scenes, taverns, workshops, dragon encounters

### ✨ **Rim Lighting / Silhouette**
Back-lighting creates glowing edges on dark figures.
```
Prompt fragment: "dramatic rim lighting, backlit silhouette, glowing edges, figure against bright background, halo effect"
```
*Use for:* Dramatic entrances, mysterious figures, dragon reveals

### 🌧️ **Volumetric / Atmospheric**
Light catches particles in the air — fog, rain, smoke, steam.
```
Prompt fragment: "volumetric lighting, god rays through fog, light beams through rain, atmospheric haze, smoke catching light"
```
*Use for:* Outdoor scenes, industrial areas, mystical moments

### 🌆 **High/Low Key Mixing**
Combining bright and dark areas in the same frame for maximum contrast.
```
Prompt fragment: "high contrast lighting, pools of light in darkness, mixed high and low key, selective illumination, noir contrast"
```
*Use for:* Complex scenes, establishing shots, narrative moments

### Lighting Don'ts
- ❌ Flat, even lighting (kills the noir)
- ❌ Bright daylight without shadow play
- ❌ Multiple soft sources (loses drama)
- ❌ Pure white light (always tint slightly)

## 6. Model Recommendations

### 🥇 **Midjourney v6+** — Best Overall
- **Strengths:** Composition, lighting, atmosphere, style consistency
- **Best for:** All DragonPunk Noir scenes, especially establishing shots
- **Settings:** Use `--style raw` for grittier results, `--ar 16:9` or `--ar 21:9` for cinematic
- **Tips:** 
  - Use `--no [negatives]` for exclusions
  - Chain with `--sref` style references for consistency
  - `::2` weight on "dragonpunk noir" to emphasize

### 🥈 **Stable Diffusion XL + Fine-Tunes**
- **Recommended Models:**
  - **DreamShaper XL** — Excellent fantasy-realism blend
  - **Juggernaut XL** — Strong cinematic quality  
  - **RealVisXL** — Grounded realism with stylization
- **Best for:** Character portraits, detailed close-ups
- **Tips:**
  - Use LoRAs for specific lighting (film noir LoRAs exist)
  - CFG 7-9 for balanced adherence
  - Add "masterpiece, best quality" to start (SDXL responds to this)

### 🥉 **DALL-E 3**
- **Strengths:** Prompt adherence, detail, accessibility
- **Limitations:** Less control over specific style, can trend bright
- **Best for:** Quick concepts, specific scene descriptions
- **Tips:**
  - Be very explicit about darkness and noir lighting
  - Describe what you DON'T want in natural language
  - Request "cinematic film still" framing

### **Flux.1 Pro**
- **Strengths:** Excellent composition, prompt following
- **Best for:** Complex scenes, multiple elements
- **Tips:** 
  - Describe exhaustively — Flux rewards detail
  - Specify lighting explicitly

## 8. Advanced Techniques

### Style Consistency Across Series
When creating multiple images for the same project:
1. Create a "style anchor" image first
2. Reference it with style-reference features (Midjourney `--sref`, SDXL style LoRAs)
3. Maintain same lighting descriptors across prompts
4. Keep color palette explicit and consistent
5. Use same negative prompt block

### Blending Fantasy and Tech
The key to DragonPunk Noir is **integration, not juxtaposition**:
- ❌ "Dragon standing next to a computer"
- ✅ "Dragon's scales interfacing with holographic display, neural connection visible"
- ❌ "Medieval castle with neon signs"
- ✅ "Gothic stonework evolved over centuries, original gargoyles now housing security cameras, new holographic guild banners floating where cloth once hung"

### Lighting for Mood
| Mood | Primary Light | Secondary Light | Shadow Density |
|------|--------------|-----------------|----------------|
| Mysterious | Single source, off-frame | Distant neon ambient | Very heavy (80%+) |
| Dangerous | Red/amber practicals | Cyan neon rim | Heavy (60-70%) |
| Melancholy | Dim ambient | Warm window glow | Medium-heavy (50-60%) |
| Powerful | Dramatic uplighting | Gold accent | Medium (40-50%) |
| Chaotic | Multiple moving sources | Fire/explosion | Irregular, shifting |

### Species Rendering Notes
- **Dragons (true form):** Emphasize scale texture, eye luminosity, breath effects. Scales should catch light like gems.
- **Dragons (humanoid form):** Subtle tells—unusual eye color, scale patches, too-perfect features, unsettling stillness.
- **Humans:** Weathered, augmented, marked by hard lives. Rarely pretty, often striking.
- **Other species:** Ground in noir tropes—orcs as muscle, elves as fixers, etc.—then subvert expectations.
